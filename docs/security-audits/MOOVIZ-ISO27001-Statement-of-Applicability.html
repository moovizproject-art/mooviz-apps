This Statement of Applicability (SoA) covers ISO 27001:2022 Annex A controls (A.5 through A.8) for the MOOVIZ delivery platform. Each control is assessed for applicability based on the platform's serverless cloud architecture (Firebase/GCP), mobile-first design, and the security audit findings remediated in March 2026. Physical controls (A.7) are largely N/A due to the fully serverless, Google-managed infrastructure.

A.7 -- Physical Controls
